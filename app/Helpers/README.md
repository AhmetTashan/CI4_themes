#  views_helper.php  dosyası içeriği

App\Views klasörünün içerisinde **components** adlı klasörümüzü oluşturalım. Daha sonra içerisine **header.php** ve **footer.php** dosyalarını oluşturalım.

Yani Klasör yapımız.

```
+ App
  + Views
    + components
      - header.php
      - footer.php
```

şeklinde olacaktır.
